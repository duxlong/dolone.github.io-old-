黑苹果问题

黑苹果 2019/01/27

1. 论坛寻找答案

  • 远景论坛：国内黑苹果论坛，新注册需要邀请码，比较麻烦。

  • 黑果小兵的部落格：黑苹果高手的博客，很多高端技术，目前好像也在B站发视频了。

  • Tonymacx86：国外黑苹果论坛，提供各种工具，也有很多帖子值得参考，不知道是不是黑苹果的起源网站。

2. DP 输出不了声音

  • 添加驱动：Lilu，AppleALC

3. 启动时加载界面（苹果标志）特别慢，要2分钟左右才能进入系统

  • 重装系统后正常，30秒内进入系统

4. CPU 不会睿频，始终以最低频率 1.6GHz 运行

  • 以下目录中，很多 plist 文件，自己在 SMBIOS 中设置的 board-ID ，需要与此处的 plist 文件名有一个匹配

  • 以上只是我个人推测，我在修改 SMBIOS，iMac12，2 改为 iMac14，2 之后，就可以正常睿频了

5. USB 不工作：待解决

6. 启动时偶尔出现禁止标志，重启后又正常：待解决
